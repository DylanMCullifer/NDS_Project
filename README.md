# Alcohol Abuse Effects on EEG Data

Authors: Will Kersh, Dylan Cullifer, Isabell Cook

## Dataset 
Kaggle EEG-Alcohol from https://www.kaggle.com/datasets/nnair25/Alcoholics


## How to use code:
1. Go to https://www.kaggle.com/datasets/nnair25/Alcoholics
2. Download zip file 
3. Extract to a folder in this repo called `eeg+database`
4. Run `NDS_Project.ipynb`

## Package Dependencies
`pip install pandas mne`
